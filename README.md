## Лабораторная работа №5 
## Задача 2 - Дерево выражений

### Описание

Дан математический пример в **постфиксной форме** (обратная польская запись.). Необходимо построить **дерево выражения**, где листья — операнды (целые числа), а внутренние узлы — операторы `+`, `−`, `*`, `/`. После построения дерева нужно вычислить значение выражения с помощью рекурсии.

### Описание задания

Реализовать функции:
- `build_expression_tree(tokens)`: построить дерево по списку токенов постфиксной записи.
- `evaluate(root)`: рекурсивно вычислить значение выражения, расположенного в дереве.
- Добавить поддержку оператора возведения в степень `^`
---
Алгоритм сначала строит дерево выражений из списка токенов с помощью стека, где числа становятся листьями, а операторы (+, -, *, /, ^) связывают извлеченные из стека поддеревья в единую структуру, сохраняя правильный порядок операндов. Затем функция evaluate рекурсивно обходит получившийся куст от корня к листьям, спускается до чистых значений и последовательно выполняет над ними математические операции снизу вверх.
